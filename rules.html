Reglas de la Comunidad

REGLAMENTOS Y NORMATIVAS

Última actualización: 31 de Enero, 2026

1. Conducta y Respeto

MuzicMania es un espacio para todos. No se tolerará el acoso, discursos de odio, discriminación o toxicidad en ninguna de sus formas. El respeto mutuo es la base de nuestra comunidad.

2. Juego Limpio (Fair Play)

Está estrictamente prohibido el uso de bots, macros, scripts externos o cualquier software que automatice las pulsaciones. La manipulación de memoria para alterar puntuaciones resultará en un Baneo Permanente.

3. Identidad de Usuario

Los nombres de usuario no deben ser ofensivos, contener publicidad no autorizada o suplantar la identidad de miembros del equipo, moderadores o figuras públicas.

4. Seguridad y Bugs

Cualquier intento de explotar bugs para beneficio propio o de terceros está prohibido. Si encuentras un fallo de seguridad, debes reportarlo inmediatamente a través de los canales oficiales.

5. Multicuentas y Boost

No se permite el uso de múltiples cuentas para manipular el Leaderboard o realizar "boosting". Cada jugador debe competir bajo una única identidad para garantizar la transparencia.

6. Privacidad y Doxxing

Está prohibido compartir información personal de otros usuarios sin su consentimiento expreso. Respetamos la privacidad de cada miembro; el "doxxing" conlleva la expulsión inmediata.

7. Resoluciones de Moderación

Las decisiones de los moderadores son finales en casos de trampas evidentes. Para otras infracciones, se podrá solicitar una revisión via soporte, pero la disciplina se mantiene firme.
