# Guidelines / Directrices

### [English]
- Use high-quality audio setups for the best experience.
- Maintain a stable internet connection for leaderboard synchronization.
- Character height and visibility settings can be adjusted in the settings menu.
- Report any bugs or visual glitches via the support channel.

---

### [Español]
- Usa equipos de audio de alta calidad para la mejor experiencia.
- Mantén una conexión a internet estable para la sincronización de las tablas de clasificación.
- Los ajustes de altura del personaje y visibilidad se pueden modificar en el menú de ajustes.
- Informa cualquier error o fallo visual a través del canal de soporte.
